# NADI — Navigating Assets, Dependents and Financial Insights

**Mapping Family Financial Readiness for a Resilient Future**

Developed by the **Faculty of Law (FUU), Universiti Kebangsaan Malaysia (UKM)**

---

## Overview

NADI is a fully static educational web application that assesses family financial readiness through integrated analysis across five dimensions:

1. Family & Dependent Profile
2. Income & Employment Structure
3. Asset Portfolio
4. Liability & Debt Position
5. Financial Protection & Estate Planning

The platform computes a **NADI Family Financial Readiness Index** (0–100) and generates a structured report with personalised analysis and recommended actions.

---

## Deployment (GitHub Pages)

1. Upload all four files to a GitHub repository:
   ```
   index.html
   style.css
   script.js
   README.md
   ```
2. Go to **Settings → Pages → Source → main branch / root**
3. Your site will be live at `https://<username>.github.io/<repo-name>/`

No build step, no dependencies, no backend required.

## Technical Stack

- **HTML5** — Semantic markup, accessible form controls
- **CSS3** — Custom properties, Grid, Flexbox, responsive design
- **Vanilla JavaScript** — Rule-based analytics engine, live calculations, Chart.js integration
- **Chart.js 4.4** (CDN) — Asset/liability bar chart, income doughnut chart
- **Google Fonts** (CDN) — Inter (body), Playfair Display (headings)

No frameworks. No backend. No authentication. No data is transmitted or stored.

---

## Analytics Engine

The NADI analytics engine uses a deterministic rule-based scoring system across five dimensions, each scored out of 20 points:

| Dimension | Max Score | Key Factors |
|---|---|---|
| Family Dependency | 20 | Dependents, dual income, marital status |
| Financial Risk | 20 | DTI ratio, emergency fund, employment stability |
| Protection Adequacy | 20 | Takaful coverage, MRTT, medical protection |
| Financial Continuity | 20 | Liquid asset coverage, takaful replacement, dual income |
| Wealth Preservation | 20 | Hibah, Wasiat, net worth |

**Total NADI Index: 0–100**

| Score | Category |
|---|---|
| 80–100 | Resilient |
| 60–79 | Developing |
| 40–59 | Vulnerable |
| 0–39 | Critical |

## Privacy

All data entered into NADI is processed **entirely within the user's browser**. No data is transmitted to any server, stored in any database, or shared with any third party.
